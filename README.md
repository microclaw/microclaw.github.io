# MicroClaw Website

This folder contains the MicroClaw docs website (Docusaurus), published at https://microclaw.ai.
It includes docs + blog content and can be developed/deployed independently from the Rust bot.

## Install

```sh
npm install
```

## Local development

```sh
npm start
```

## Build

```sh
npm run build
```

## Serve build output

```sh
npm run serve
```
